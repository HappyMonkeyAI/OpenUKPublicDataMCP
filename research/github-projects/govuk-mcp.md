# GovUK-MCP

- URL: https://github.com/Stealth-Labs-LTD/GovUK-MCP
- Type: Existing general UK public-data MCP server
- Stack: Python MCP server
- Reported scope: 33 tools over Companies House, Parliament, NHS, Transport, postcodes, food hygiene, flood warnings, police crime, courts, charities, CQC, legislation, bank holidays, GOV.UK search.
- Auth: Companies House and EPC required; TfL optional; many tools no-key.
- Why it matters: proves demand and provides a useful baseline for source selection.
- What to cherry-pick: broad categories, user-friendly prompts, widget idea for visual clients.
- What to avoid: positioning as comprehensive while needing keys for some headline tools; hobby/demo reliability posture.
